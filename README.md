# Impact of Missing Data on Sample Size Requirements for Developing Clinical Prediction Models

Repo for the paper entitled "Impact of Missing Data on Sample Size Requirements for Developing Clinical Prediction Models", which is currently in-preparation for submission.

The repo contains the coding scripts and results from the simulation study described in the paper as follows:

## Code sub-folder

This contains the R scripts used in the simulation study. The running of the simulation relied heavily on computational facilities at the University of Manchester.

## Outputs sub-folder

This contains the .RDS files of the results of the simulation, as described in the paper. Also contains the results of the two practical case studies that are described in the paper.
